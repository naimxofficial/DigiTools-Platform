# 🛠️ DigiTools Platform

> **Your all-in-one platform for digital tools — browse, cart, and subscribe.**

DigiTools Platform is a modern React-based web app where users can explore digital tools and products, manage a shopping cart, view pricing plans, and subscribe — all in a fast, responsive single-page experience.

🌐 **Live Site:** [https://digitools-platform-naimxofficial.netlify.app/](https://digitools-platform-naimxofficial.netlify.app/)

---

## ✨ Features

| Feature | Details |
|--------|---------|
| 🏠 **Hero Section** | Eye-catching landing banner with key messaging |
| 📊 **Stats Section** | Platform stats to build trust with visitors |
| 🔧 **Tools Section** | Highlights the core digital tools offered |
| 🛒 **Products & Cart Tabs** | Toggle between browsing products and viewing your cart — with live item count on the Cart tab |
| ➕ **Add to Cart** | Add digital products to cart with toast notifications via React Toastify |
| 🗑️ **Cart Management** | View and manage cart items in a dedicated Cart tab |
| 🚀 **How It Works (Steps)** | Step-by-step guide section for new users |
| 💰 **Pricing Section** | Subscription plan comparison |
| 📬 **Ready to Subscribe CTA** | Conversion section to push users toward signing up |
| 🔔 **Toast Notifications** | Real-time feedback on cart actions using React Toastify |
| 📱 **Fully Responsive** | Optimized across mobile, tablet, and desktop |
| 🎨 **Purple Gradient Theme** | Consistent `#4f39f6` to `#9514fa` gradient across interactive elements |
